# Проект: генерации документации Java через Docker

Проект выполнен в рамках курса Java Core.
Цель: автоматизировать создание Javadoc-документации в изолированной среде.

## Состав проекта:
- `src/` — исходный код приложения.
- `Dockerfile` — описание образа для генерации документации.
- `docker-compose.yml` — файл для удобного запуска контейнера.

## Инструкция по запуску:
1. Убедиться, что установлен Docker и Docker Compose.
2. Клонировать репозиторий.
3. Откройте терминал в папке проекта.
4. Выполните команду:
   ```bash
   docker-compose up --build
5. После завершения работы контейнера, готовая документация появится в папке mydocumentation/.
6. Открыть файл mydocumentation/index.html в браузере для просмотра.
